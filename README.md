# AILifeChatbot
The AI Life Dashboard is an AI-powered web app that unifies student tools into one platform. It uses multiple AI agents, smart routing, and real-time mapping with OpenStreetMap and OSRM. Features include quiz generation, progress tracking, and notes synthesis, all within a modern, user-friendly interface.

## 🚀 Key Features

- 🤖 AI Agents System  
  Multiple intelligent agents handle study help, career guidance, and productivity tasks.

- 🔀 Smart Auto-Routing  
  Automatically detects user intent and routes queries to the correct AI agent.

- 🗺️ Real-Time Route Planning  
  Uses OpenStreetMap and OSRM for navigation and location-based assistance.

- 📚 Quiz Generation  
  Generates dynamic MCQs with answers and explanations.

- 📊 Progress Tracking  
  Tracks learning performance and highlights weak areas.

- 🧠 Mistake Analysis  
  Identifies errors and provides step-by-step corrections.

- 📝 Notes & PDF Generator  
  Converts study content into structured notes and downloadable PDFs.

- 🎨 Modern Glassmorphism UI  
  Clean, responsive, and visually appealing interface.
